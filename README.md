•	Project Name: PostArt<br /><br />
•	Project Applcation:<br />
  •	In PostArt project,a publicly accessible registered domain is used where new users can create account and can login through username     and password.<br />  
  •	After providing the correct username and password, a user can upload, view, download, update their created art like paintings,      
    sketches, and so on.<br /> 
  •	Through PostArt users will also be able to update the file if any changes are done.<br /> 
  •	User will be able to see details like first name, last name, file uploaded time and file updated time.<br /> 
  •	User also can delete the file if it’s no longer needed.<br /><br />
•	Techologies Used:<br />
  •	AWS Components:- EC2, ELB, Lambda, AutoScaling Group, RD -MySQL, Cloudfront,S3, S3 Transfer Acceleration, R53,        
    Cloudwatch,SNS.<br /> 
  •	AWS sdk for php.<br />
  •	Apache Server.<br />
  •	Putty.<br />
  •	WinSCP.<br /><br />
•	Features List:<br />
  •	Users will be able to perform the following functions:-<br />
    •	Upload a file (upto 10 mb) to the S3 bucket.<br />
    •	View the file from the S3 bucket.<br />
    • List the files from the S3 bucket.<br />
    •	Downlaod the file from the S3 bucket.<br />
    •	Update the file from the S3 bucket.<br />
  •	Users will also be able to see the following details with the files from the S3 Bucket:-<br />
    •	User’s first name.<br />
    •	Users’s last name.<br />
    •	File Upload time.<br />
    •	File Updated time.<br />
